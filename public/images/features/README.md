# Feature Images

Este diretório contém as imagens utilizadas na seção "Features that make the difference" da landing page.

## Estrutura de Imagens

Para a feature "Facebook Ads Integration" (primeira feature), coloque as seguintes imagens:

- `facebook-ads-dashboard.png` - Imagem 1: Dashboard com métricas e gráficos
- `facebook-ads-admin.png` - Imagem 2: Página de Admin Management

### Resolução Recomendada

- **Tamanho:** 800x800px (mínimo) ou superior
- **Formato:** PNG ou JPG
- **Qualidade:** Alta resolução para suporte a zoom

### Outras Features

As outras features também precisam de imagens (placeholders por enquanto):

- `metrics-1.png`, `metrics-2.png` - Automatic Metrics
- `ai-1.png`, `ai-2.png` - AI for Decisions
- `automation-1.png`, `automation-2.png` - Complete Automation
- `profit-1.png`, `profit-2.png` - Profit Analysis
- `secure-1.png`, `secure-2.png` - Secure Data

## Como Adicionar Imagens

1. Coloque as imagens neste diretório (`public/images/features/`)
2. Use os nomes de arquivo especificados acima
3. As imagens serão automaticamente carregadas na página

## Notas

- As imagens são carregadas com `loading="lazy"` para melhor performance
- Se uma imagem não for encontrada, um placeholder será exibido automaticamente
- As imagens suportam zoom ao clicar (clique na imagem para ampliar)
